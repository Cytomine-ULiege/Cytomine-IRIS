Project Overview Table

The project overview table shows the following properties for each project available to you:

  • Name: the name of the Cytomine project
  • Created: the creation date
  • # Images: the total number of images in this project
  • Mode: Blind Mode includes undisclosed image names and hidden annotation information, which is revealed in Regular projects
  • Action: actions available for the project
  • Info: displays general information about the project, such as contact information for support issues
Per default, the projects are sorted by creation date, such that the most recent project is displayed on top of the table. Click on the column titles to change the sort order in the table. You can also search for a specific project name using the text field in the Name column. If the project has recently been added, its entry exibits a New! label next to the creation date.

Opening a Project

Open a project by clicking on the button in the row of the project's name. If the button is disabled , the project is not (yet) available to you or does not have any images to be examined.

Update the Project List

Update the project list to reflect the latest changes by pressing Refresh at the top of the page.
