Project Overview Table

The project overview table shows the following properties for each project available to you:

  • Name: the name of the Cytomine project. A label indicating Locked indicates that you do not have access to this project. Hover the label to receive more information.
  • Created: the creation date
  • # Images: the total number of images in this project
  • # Annotations: the total number of annotations in this project
  • Mode: Blind Mode includes undisclosed image names and hidden annotation information, which is revealed in Regular projects
  • Actions: actions available for the project
  • Info: displays general information about the project, such as contact information for support issues
Per default, the projects are sorted by creation date, such that the most recent project is displayed on top of the table. Click on the column titles to change the sort order in the table. You can also search for a specific project name using the text field in the Name column. If the project has recently been added, its entry exhibits a New! label next to the creation date.

Available Actions

Select one of the available actions for your task. Any user that has no access to a project can request it from the project coordinators using the Request Access link. Once the user has access, this button is not visible to them. Some other buttons may be deactivated according to your current rights (i.e. being a coordinator for that project or not). The available actions are detailed in the following sections.

Opening a Project

Open a project by clicking on the button in the row of the project's name. If the button is disabled , the project is not (yet) available to you or does not have any images to be examined.

Request Access

If the project is Locked, you can request access by clicking on the dropdown button and selecting .
A dialog will guide you through the request process. The request will be sent to the project coordinators, who will eventually grant access to the project. You can submit a short message to the coordinators as well. Once the coordinator granted you access, you will be notified via email.
If there is no coordinator set yet for that project, you can request to become one yourself.
Please note: If you already have access to the project, this button will not be visible to you.

Coordinator Request

If you want to become a coordinator of the project, e.g. to alter settings and view project statistics, select from the dropdown.
A dialog will guide you through the request process. The request will be sent to the administrator of the IRIS host, who will eventually authorize you as coordinator of the project. You can submit a short message to the administrator as well. Once you are authorized, you will be notified via email.

Settings (Coordinators only)

Opens the project settings and lets you edit project and image access rights for all users. Furthermore, the labeling progress for each user can be updated manually from the Cytomine host.

Statistics Dashboard (Coordinators only)

Opens the project statistics dashboard and reveals user statistics and agreement visualizations on the annotations.

Update the Project List

Update the project list to reflect the latest changes by pressing at the top of the page.

Keyboard Shortcut List
