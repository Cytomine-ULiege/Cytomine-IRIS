Access Management

Generally, you can enable or disable particular projects and images for particular users, but this requires you to be a project coordinator. As a project coordinator you are able to view and alter access rights to the project and each image per user and authorize other users as project coordinator as well.

All project users are listed in a table in the page, and a short summary is displayed right above the table. If there is just a single project coordinator in this project, the number is replaced with the name of that user, otherwise, hovering the number will display all coordinators of the project (also if they are on different pages of the user table).
  • User ID: the Cytomine user ID
  • Last Name: the user's last name
  • First Name: the user's first name
  • Username: the user's user name
  • Enabled: a flag indicating whether or not the user has access to this project on this IRIS host
  • Coordinator: a flag indicating whether or not the user is a coordinator of this project on this IRIS host
  • Actions: : this will open a modal dialog, displaying all images in the project, where you can enable and disable each image for the user.
Hint: always try to refresh the table before making changes!
Project Coordinator
In order to establish a person as a coordinator, select the user's check box in the column Coordinator. If you authorize someone as coordinator, the project will automatically be enabled for that user. Moreover, the user will receive an email indicating that coordinator rights are now available for that project.
Similarly, deselect the check box, if you want to revoke the user's rights as coordinator.

Please note: You cannot remove yourself as a project coordinator.

Project Access
Project access is granted by selecting the user's check box in the column Enabled. Disable the project for a user by deselecting the check box.

Please note: You cannot disable the project for yourself.

Image Access
Open the images for a user by selecting from the table. This will open a modal dialog, where you can enable and disable the images using the check box from the rightmost column. The summary at the top of the table will be updated accordingly.

Labeling Progress Update/Synchronization

From the tabs, select Synchronization to view the corresponding settings.

Computing the labeling progress for each user in an image is a rather expensive operation and is just executed automatically at very specific moments during a workflow in the application. Nevertheless, as a coordinator, you can trigger synchronization manually either for all users (using the big red button ;-)) or for individual users by pressing .

While synchronization is in progress, the buttons will be deactivated and become reactivated when the procedure is finished.
