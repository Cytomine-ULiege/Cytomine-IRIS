Applying a Filter

The annotation gallery lets you filter all available annotations by the terms you already assigned in the labeling view. It is a tool for validating your assignments, such that outliers can be identified easily by visual inspection. Per default, all images are selected, which lets you compare the same term in the entire project. On the left hand side the Filter Panel hosts all available filter options.

Term Filter
All terms are available in the ontology tree, which is associated with the project. The term tree reflects the hierarchy of the ontology, where each node is collapsible and each leaf is checkable. Hide the tree using the hide button.

Check the check box next to the term description in order to filter for the term.

Image Filter
The images associated with the project can be selected arbitrarily. If no image is selected, there will be a notification. Show all images using the show button.

If you want to check or uncheck all items, use the convenient buttons at the bottom of the term and image filter options.

Update the Filter
Update the gallery with the current filter by pressing Refresh filter at the bottom of the page.

Filtered Terms

Each selected (filtered) term will be displayed in a dedicated panel (see example below). If the number of annotations having the filtered term exceeds a particular limit, they are split up in pages. The pages can be accessed using the paginator controls below the panel's header. In the image below, the first page is selected. You can select all elements from one page using the button from the top right corner of the panel. Similarly, all elements of the page can be de-selected using the button.

Correcting Term Assignments

If you require to alter the term of annotations, there are three options to do that:

  • Labeling view: use the context menu described below and select the option "Assign another label...". You will be taken to the labeling view and can examine the annotation in the context of the entire slide
  • Drag&Drop: only for altering the term on a single annotation
  • Selective Assignment: mainly for altering the term on multiple annotations
Each annotation element has a few options you can use to alter the terms:
  • Checkbox: Add or remove the annotation from a multi selection
  • Drag handle: Element you can use to drag the annotation and drop it on another term
  • Context menu: Popup menu, which lets you assign a new term or reveal the annotation in Cytomine

  1. Choose a term, some images and filter the annotations.
  2. Move the mouse over the drag handle of the annotation you want to re-assign and press the left mouse button.
  3. Keep the left mouse button pressed and drag the handle over the term to be assiged in the term tree on the left side. The area where you can drop the handle will be highlighted once you position the handle over it.
  4. Drop the handle on the highlighted area. You will notice that the annotation vanishes from the previous position and re-appears in the target group, and that the total number of annotations gets updated.

Change the label of a single item via Drag&Drop on the Panel Header

Change the label of multiple items using selective assigment

Watch the short tutorial video by clicking on the image.

  1. Choose a term, some images and filter the annotations.
  2. Select all annotations (also from different groups) you want to assign the same term to using their Checkbox. Once at least one item is selected, the button in the toolbar (bottom of the page) gets enabled.
  3. From the dropdown list in toolbar, select the label to assign to the selected annotations. The button is being enabled once a valid label has been chosen.
    Be careful when assigning the same term to multiple annotations!
  4. If the button got enabled, you can now click it and assign the selected label to all selected annotations. The total number of annotations in the headers get updated automatically and the selection of the annotations is cleared.

Removing Terms

Terms can be removed from the annotations the same way you re-assign terms. The only difference is that you select "no term assigned" from the dropdown list in the bottom toolbar, or drop the drag handle on  No term assigned in the term tree.
