Start Image Labeling

Start labeling the annotations of an image by choosing "Start Labeling" from the Actions column in the table.

An image can be selected for labeling, when there is at least one annotation available. If there are no annotations for that image, the button will be disabled.

Progress Filtering

The progress of labeling is viewed in a tooltip, when you let the mouse rest on the progress field.

The images can be filtered according to their progress:
  • If "Show all" is selected, any progress will be displayed.
  • If "Hide completed" is selected, only images with incomplete label assignments are listed in the table.
So if you are looking for an image, consider that the filter option may be enabled.

Viewing Annotation Gallery

All annotations can be viewed in an annotation gallery. In order to view all annotations and have some filters available, choose "View Annotation Gallery" from the dropdown box.

Exploring the Image in Cytomine

If required, the particular image can be viewed in the Cytomine Explorer. A new window will be opened for the view.
