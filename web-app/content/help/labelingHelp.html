Labeling Annotations

On this page you can assign labels to and and remove them from available annotations. Navigate through all annotations using the buttons « Previous and Next » or use the keyboard shortcuts p and n respectively.

Once you have identified the desired annotation, simply choose one of the terms from the list on the right hand side and assign the label by clicking on the button, which is named accordingly.
If you want to remove an existing term, choose Remove label from the top right corner of the labeling panel. This button is disabled, if the annotation does not have any term to be removed.
The currently assigned label is displayed right above the view. The label turns green if there is a term assigned, and stays orange, if there is no term assigned.
At the top of this page, the total number of remaining unlabeled annotations is displayed along the labeling progress within the image. As you progress with assigning terms, the progress bar will climb up to 100%. Exploring the Annotation Context

The 'exploration' view provided in the left part of this page displays the annotation within the immediate surroundings in the slide. You can zoom in and out using the control elements in the upper left corner or using the mouse wheel. Change the view as required by clicking and dragging the slide (the cursor turns to a 'move' cursor).

  • Reset view takes you back to the initial view, zooms in/out optimally and centers the annotation
  • Pan to annotation keeps the current zoom factor but centers the annotation
  • View in Cytomine opens a new window/tab and lets you explore the entire slide in a larger view on the Cytomine website

Filter Completed Annotations

The option Hide completed in the upper left corner allows for filtering already labeled annotations and the navigation will adapt automatically. Whether or not you are displaying just the unlabeled instances is displayed by a filtered sign next to the current annotation number.

Filter Labels

You can also filter labels by typing in e.g. a prefix in the text boxes right below the headers. Clear the filter and show all terms again using the button Clear filter in the bottom left corner.
